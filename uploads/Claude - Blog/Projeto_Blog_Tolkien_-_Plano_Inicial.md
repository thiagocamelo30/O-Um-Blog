# Plano Inicial — Blog sobre o Universo de Tolkien

## 1. Sugestões de nome para o blog

Critérios: remetem à Terra-média, fáceis de lembrar/pronunciar em português, e evitam usar "Tolkien" no nome (reduz risco de questões de marca e ainda assim ranqueia bem, já que o conteúdo deixa o tema claro).

1. **Portal de Arda**
   - "Arda" é o nome do mundo onde a Terra-média existe — termo reconhecível pelos fãs, mas ainda nicho o suficiente para não ter concorrência forte de marca.
   - Domínios a checar: portaldearda.com.br / .com

2. **Crônicas da Terra-Média**
   - Nome direto, fácil de lembrar, comunica claramente o nicho (importante para CTR no Google). "Crônicas" também sugere conteúdo de lore/história, o que ajuda no SEO de cauda longa.
   - Domínios a checar: cronicasdaterramedia.com.br / .com

3. **O Um Blog** (trocadilho com "Um Anel")
   - Nome curto, memorável, com um trocadilho que fãs reconhecem na hora ("Um anel para a todos governar"). Curto = bom para branding e redes sociais.
   - Domínios a checar: oumblog.com.br / oumblog.com

> Recomendo verificar a disponibilidade real dos domínios (.com e .com.br) e do @ nas redes sociais antes de decidir — eu não tenho acesso a ferramentas de verificação de domínio neste momento, mas qualquer registrador (Registro.br, GoDaddy, Namecheap) resolve isso em segundos.

**Lógica do cronograma:**

- Os títulos de lore foram escolhidos por terem alto volume de busca e baixa cobertura em português (oportunidade de ranquear mais rápido que termos já saturados como "quem é Aragorn").

## 3. Estratégia de SEO — "Qual a ordem certa para ler os livros de Tolkien?"

### Intenção de busca
Mista: parte informacional ("qual ordem ler"), parte transacional implícita (quem pesquisa isso geralmente está perto de comprar os livros). Isso justifica colocar links de afiliado de forma natural ao longo do texto, não só ao final.

### Estrutura recomendada (H1/H2/H3)

- **H1:** Qual a Ordem Certa para Ler os Livros de Tolkien? Guia Completo 2026
- **Introdução (100–150 palavras):** responda a pergunta principal já no primeiro parágrafo (ex: "Não existe uma única ordem certa — existem duas abordagens..."), para aumentar a chance de aparecer em featured snippet.
- **H2: Ordem de publicação x ordem cronológica — qual a diferença**
- **H2: Ordem recomendada para quem está começando agora**
  - H3: 1. O Hobbit
  - H3: 2. O Senhor dos Anéis (trilogia)
  - H3: 3. O Silmarillion (depois, não antes)
- **H2: Ordem cronológica da história (para quem já leu e quer entender a linha do tempo)**
  - Lista numerada com os eventos/era de cada obra
- **H2: Tabela comparativa de edições disponíveis** (aqui entram os links de afiliado)
  - Tabela: Edição | Editora | Preço aproximado | Ideal para | Botão/Link "Ver na Amazon"
- **H2: Perguntas frequentes (FAQ)**
  - "Preciso ler o Silmarillion para entender O Senhor dos Anéis?"
  - "Posso ler O Hobbit depois da trilogia?"
  - "Qual a melhor edição para iniciantes no Brasil?"
  - Usar marcação de Schema FAQPage — aumenta a chance de rich snippet no Google.

### Elementos técnicos de SEO
- **Title tag:** "Qual a Ordem Certa para Ler os Livros de Tolkien? (Guia 2026)" — manter abaixo de 60 caracteres.
- **Meta description:** incluir a palavra-chave principal + CTA, ex: "Descubra a ordem ideal para ler os livros de Tolkien, da estreia com O Hobbit ao Silmarillion, com dicas de edições para comprar."
- **URL slug:** /ordem-para-ler-livros-tolkien (curto, com a keyword principal)
- **Schema markup:** Article + FAQPage (e ItemList se a tabela de edições for destacada)
- **Links internos:** linkar para os artigos 2, 4 e 5 do cronograma sempre que mencionar personagens/eventos (ex: ao citar o Silmarillion, linkar para o artigo sobre Valar e Maiar)
- **Links externos de afiliado:** usar `rel="sponsored nofollow"` nos links da Amazon (exigência do Google para conteúdo de afiliados)
- **Imagens:** capas dos livros com `alt text` descritivo (ex: "capa da edição Harper Collins de O Senhor dos Anéis") — ajuda no Google Imagens, fonte extra de tráfego para este tipo de artigo
- **Atualização:** este artigo deve ser revisado a cada poucos meses para atualizar preços/edições — artigos de "guia de compra" perdem ranking se ficarem com informação de preço desatualizada
